Coherence Lens

Compositional TDA. Beyond Vietoris-Rips: simplicial complexes via embedding model as compositional oracle.

Key Finding

comp_ratio ~ 0.70 across the 14-month archive. 30% of VR-candidate triples fail the compositional test, confirming that Vietoris-Rips flattens the Kan-depth hierarchy.

Methodology

For each fibre (monthly archive chunk), build a compositional complex:

  1. Compute pairwise distances between conversation embeddings
  2. For candidate triples within VR threshold, test compositional coherence: embed(a+b+c) vs mean(embed(a), embed(b), embed(c))
  3. Only add 2-simplex if composition test passes (deviation < threshold)
  4. Compute persistence, Betti numbers, bottleneck distance

Inter-fibre bottleneck distance = Presence metric. Four-mode classification from persistence profiles.
